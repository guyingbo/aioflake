Unique id generator inspired by twitter's snowflake.

## Usage

```python
import asyncio
import aioflake


async def go():
    flake = aioflake.Flake()
    ident = await flake.next()
    print(ident)
    idstr = aiflake.urlsafe(ident)
    print(idstr)


loop = asyncio.get_event_loop()
loop.run_until_complete(go())
loop.close()
```
